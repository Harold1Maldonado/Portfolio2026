# Harold Maldonado Portfolio

Portfolio web built with:
- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- Replace the placeholder project data in `src/App.jsx`.
- The embed section uses `iframe`. Some sites block embedding with `X-Frame-Options`; if that happens, replace the iframe with an image/mockup and keep the external link.
- You can later split the content into components if you want a cleaner architecture.
